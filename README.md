Steps to run locally:

1. download the project
2. unzip it
3. cd -> 'name of project folder'
4. npm i
5. php artisan serve

\*\*make sure that sql is active as the data will be saved in a sql DB

db login settings:

DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=onlineshop
DB_USERNAME=root
DB_PASSWORD=

runs on port 8000
